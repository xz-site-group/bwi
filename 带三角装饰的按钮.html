CSS代码 (使用border方法):
.custom-button {
  position: relative;
  width: 200px;
  height: 48px;
  background-color: #ff0000;
  color: white;
  font-weight: bold;
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 左侧三角 */
.custom-button::before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: 0;
  height: 0;
  border-top: 24px solid #ff0000;
  border-bottom: 24px solid #ff0000;
  border-left: 10px solid transparent;
  z-index: 2;
}

/* 右侧三角 */
.custom-button::after {
  content: '';
  position: absolute;
  top: 0;
  right: 0;
  width: 0;
  height: 0;
  border-top: 24px solid #ff0000;
  border-bottom: 24px solid #ff0000;
  border-right: 10px solid transparent;
  z-index: 2;
}

实现说明

我提供了三种实现方法：

  1. 使用border方法：通过CSS的border属性创建三角形，这种方法简单直接，但需要精确计算尺寸。
  2. 使用skew变换方法：通过伪元素的skew变换创建斜角，这种方法更灵活，可以调整斜角的角度。
  3. 使用clip-path方法：使用clip-path属性直接裁剪出带斜角的形状，最现代的方法，但IE兼容性较差。

所有方法都避免了使用透明边框带来的问题，并确保了背景颜色的一致性。
